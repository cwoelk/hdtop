htop for Docker

## Building

1. Install [go](http://golang.org/)
2. Install [gb](http://getgb.io/) with `go get github.com/constabulary/gb/...`
3. `git clone git@github.com:hhdug/hdtop.git && cd hdtop`
4. `gb build`

### Alternative

Run ```./build.sh``` under Linux with Docker installed.
